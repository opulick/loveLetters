Dear GitHub,

I miss you so much, that I wish there was a clone1 of you so you could be with me all the time. You are able to read me2 so well when I am with you. I feel that there is this force that pushes3 and pulls4 at me when I am away from you for too long. My only request5 is for you not to hold anything against me when we are at our worst. I vision our future that one day we will be able to merge6 our lives together and grow old together. I think it would be funny that either you or me will be forking7 food over to each other when we are old and wrinkly. If only there was a remote8 that would allow me to pass all the hard times and pause for all the good times with you. I need a full report9 on how your trip to Thailand was.


Love,

Olivia


  1. git clone url - Clone a repository to a local working directory
  2. readme.md - a file that provides information or directions to a project
  3. git push origin master - Pushes changes from local working directory to github
  4. pull - gets and merges the changes on the remote server to a local working directory
  5. pull request - requests your changes to be accepted by original owner
  6. merge - merge the two branches
  7. fork - make a copy of repository without it affecting the original branch
  8. git remote add original url - sets up connection to a remote repository
  9. repo - the file history of a project
